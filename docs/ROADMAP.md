# EduArabic – Product Roadmap

## Phase 1 — Foundation (Current) ✅
**Goal**: Core platform operational for early institutions

- [x] Multi-tenant institution management
- [x] Role-based authentication (6 roles)
- [x] Student management + auto-generated Student IDs
- [x] Class management + enrollment
- [x] Offline attendance tracking
- [x] Learning Center (lessons, subjects, resources)
- [x] Assessment Engine (quiz, exam, practice, assignment)
- [x] AI Question Generation
- [x] Hifz progress tracking
- [x] Audio submission + Tajweed AI analysis
- [x] Parent portal (attendance, results, Hifz, fees)
- [x] Student portal (courses, assessments, certificates)
- [x] Certificate system (QR verification)
- [x] Financial management (fee structures, payments)
- [x] Announcements & communication center
- [x] Offline-first with conflict resolution

---

## Phase 2 — Growth (Q3 2026)
**Goal**: Deepen AI capabilities, improve parent engagement

- [ ] Push notifications (browser + mobile)
- [ ] PDF certificate export (server-side rendering)
- [ ] Live audio recording in-browser (no upload needed)
- [ ] Bulk student import (CSV)
- [ ] Bulk attendance upload
- [ ] Teacher performance analytics dashboard
- [ ] Automated Hifz progress reports (PDF export)
- [ ] Custom branding per institution (logo, colors)
- [ ] Student self-registration link (invite-based)
- [ ] Assessment timer with auto-submit

---

## Phase 3 — Scale (Q4 2026)
**Goal**: Enterprise readiness + multi-region expansion

- [ ] SMS notifications (Twilio integration)
- [ ] WhatsApp integration for parent updates
- [ ] Multi-language UI (Arabic RTL full support)
- [ ] Multi-currency fees (beyond GHS)
- [ ] Online fee payment (Paystack for West Africa, Stripe for global)
- [ ] Timetable management
- [ ] Live class integration (Zoom/Google Meet embed)
- [ ] Advanced role permissions (custom roles per institution)
- [ ] Data export (GDPR compliance package)
- [ ] Automated backup notifications

---

## Phase 4 — Intelligence (2027)
**Goal**: Personalized AI-driven learning

- [ ] Personalized Hifz revision schedules (AI-generated, teacher-approved)
- [ ] Predictive at-risk student detection
- [ ] Natural language parent progress reports
- [ ] Adaptive assessment difficulty
- [ ] AI-generated lesson plans from curriculum objectives
- [ ] Voice-based student feedback (Tajweed coaching mode)
- [ ] Cross-institution benchmarking (anonymized)

---

## Ongoing (All Phases)
- Performance optimization (Core Web Vitals)
- Accessibility improvements (WCAG AA compliance)
- Security audits and penetration testing
- Supabase Edge cache optimization
- PWA offline coverage expansion
